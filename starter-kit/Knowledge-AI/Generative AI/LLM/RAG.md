# RAG

Retrieval-Augmented Generation: pairing an LLM with an external index ([[Frameworks and Tools/Vector Databases/Vector Databases|vector databases]] like [[Frameworks and Tools/Vector Databases/FAISS|FAISS]], [[Frameworks and Tools/Vector Databases/Chroma|Chroma]], or [[Frameworks and Tools/Vector Databases/Qdrant|Qdrant]]) so answers can be grounded in retrieved documents.
