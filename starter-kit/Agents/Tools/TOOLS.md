# Shared Tools

Tool metadata is shared across all agents. Permission settings decide whether a tool can run; profile frontmatter does not restrict access.

- [[Agents/Tools/excel-mcp/TOOL|Excel MCP Server]] for spreadsheet automation.
- Internal vault tools are available from the app registry and appear in the Tools view.
- Registered MCP tools can be documented as `Agents/Tools/<tool-name>/TOOL.md`.

Usage history is derived from `Agent Runs/` notes.
