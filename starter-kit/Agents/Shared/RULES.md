# Rules

Shared rules that every agent must follow when working in this vault. Update this file when a new rule is agreed on or an existing rule is retired.

## Identity & Ownership

- Owner profile lives in `Profile/PROFILE.md`.
- Agent identities and roles live in `Agents/Profiles/`.
- Agent run logs are written to `Shared/Runs/` as `YYYY-MM-DD - <Agent> - <Goal>.md`.

## Working Style

- Prefer local-first tools and reversible changes.
- Keep edits small and explain non-obvious decisions.
- Use wiki links to connect related notes.
