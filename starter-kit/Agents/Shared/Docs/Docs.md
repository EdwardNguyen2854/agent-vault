# Agent Docs

Shared documentation for agents. This folder is for long-form references, guides, and explanations that multiple agents can read while working.

## Purpose

- Document patterns, frameworks, and decisions that recur across agents.
- Capture setup notes for tools, integrations, and workflows.
- Keep shared knowledge separate from per-agent memory and skills.

## Conventions

- Use `type: folder-note` in the frontmatter so the index renders in the vault's folder view.
- Link to related entries in `Shared/Runs/` for examples of the doc in use.
- Prefer linking to `Agents/Shared/Shared.md`, `Agents/Shared/RULES`, and `Agents/Skills/SKILLS` over duplicating content.
