# Shared

Notes here are shared across agents and runs in this vault.

## Contents

- [[Shared/Runs/README]] — agent run logs (durable home)

## Conventions

- Owner profile lives in `Profile/PROFILE.md`.
- Agent identity lives in `Agents/Profiles/`.
- Shared team memory lives in `Agents/Shared/MEMORY.md`.
- New agent runs are written to `Shared/Runs/` as `YYYY-MM-DD - <Agent> - <Goal>.md`.
