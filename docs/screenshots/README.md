# Screenshots

Place PNG files here with the names below. Then uncomment the `<img>` tags in `docs/screenshots/README.md` (or the README) to display them.

| File                 | Description                              |
| -------------------- | ---------------------------------------- |
| `dashboard.png`      | Main dashboard with vault stats          |
| `graph-3d.png`       | 3D knowledge graph with filter panel     |
| `editor-split.png`   | Split-pane editor with markdown preview  |
| `command-center.png` | Command center with search and shortcuts |
| `chat.png`           | LM Studio chat panel with model selector |

## Naming convention

- File names must match exactly: `dashboard.png`, `graph-3d.png`, etc.
- Recommended resolution: 1280×720 px minimum.
- Recommended format: PNG or WebP with transparency.
- Maximum file size: 2 MB per image.
